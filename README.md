# Chat (基于Web的聊天系统-模拟QQ的基本功能)

## 项目介绍

本项目是一个仿QQ基本功能的前后端分离项目。前端采用了vue.js技术栈，后端采用springboot+netty混合开发。实现了好友申请、好友分组、好友聊天、群管理、群公告、用户群聊等功能。
